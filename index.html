Operator precedence and associativity

Operator precedence determines the order in which operators are evaluated in an expression. Operators with higher precedence are evaluated before those with lower precedence.

Associativity determines the order in which operators of the same precedence are evaluated, either from left to right (left-associative) or right to left (right-associative).

Example: let result = 10 + 5 * 2; // 20, not 30, because multiplication has higher precedence than addition

3: Control Structures and Loops

Conditional statements (if, else, else if) Conditional statements allow you to execute specific code blocks based on certain conditions. The basic structure of an if statement is:

if (condition) {

// Code to execute if the condition is true

}

You can also use else to specify a code block that should be executed if the condition is false:

if (condition) {

// Code to execute if the condition is true

}if (condition) {

// Code to execute if the condition is true

} else {

// Code to execute if the condition is false

}

Loops (for, while, do-while)

Loops allow you to execute a block of code repeatedly. There are three main types of loops in JavaScript:

for: Executes a code block a specific number of times.

while: Executes a code block as long as a condition remains true.

do-while: Executes a code block at least once, and then continues executing as long as a condition remains true.
